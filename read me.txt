Project: Weather Patterns & Fire Incidents in Central Brazil
------------------------------------------------------------


Overview
---------
This project analyzes weather trends and fire incidents in Central Brazil using data from BigQuery. The scripts are structured for modular execution, ensuring easy reproducibility and clear organization.


File Structure
--------------
code/
│── 00_setup.R             # Install and load required packages
│── 01_import_data.R       # Import data from BigQuery
│── 02_clean_data.R        # Clean and preprocess data
│── 03_process_fires.R     # Process fire-related data
│── 04_process_weather.R   # Process weather data
│── 05_visualize_fires.R   # Create fire-related visualizations
│── 06_visualize_weather.R # Create weather-related visualizations
│── functions/
│   ├── helpers.R          # Utility functions for plotting and transformations
│── input/                 # Raw input data
│── output/                # Generated plots and results
│── run_all.R              # Master script to run all steps
│── README.txt             # Project overview & instructions
│── .Rproj                 # R project file for environment setup


Setup Instructions
------------------
1. **Install R and RStudio** if not already installed.
2. Open the `.Rproj` file to set up the working environment.
3. Run the `run_all.R` script to execute the full workflow.

Output Files
------------
All outputs are saved in the `output/` folder, including:
- `fires_YEAR.pdf`: Fire incident maps for each year
- `fires_heatmap_YEAR.pdf`: Fire density heatmaps
- `temperature_trend.pdf`: Trend of extreme temperature days
- `drought_trend.pdf`: Trend of drought occurrences
